前端面试总结

相关注意事项


一般面试流程

组员面 –> 组长面 –> 总监面 –> hr面

组员面

html

  • html5加了什么？
  • html语义化是什么？
  • seo怎么整？
  • html5新标签有哪些？
  • doctype？

    css

  • 居中方案
  • flex布局
  • grid布局

js

  • 闭包
  • 原形
  • this
  • settimeout
  • 可能要手写一些比较常见的函数，debounce,new,Object.create,promise,bind
  • 设计模式
  • 各种继承方法

http

  • https
  • 缓存
  • 跨域
  • 网络安全
  • 状态码

其他

  • 访问页面的过程
  • 性能优化
  • 移动端自适应(我的方案是REM布局)
  • vue源码，双向绑定等等(我有个关于虚拟dom的分享)
  • 离职原因
  • 团队协作管理
  • 算法，准备一下各种排序，背包问题，二叉树，应该够了。
  • pwa，小程序

组长面

  • 项目部署
  • 项目监控
  • 错误处理
  • nodejs
  • 职业规划

我会问的问题

  • 项目有多少人
  • 项目要做些什么
  • 项目的前景
  • 如果我进去了可能做什么
  • 前端在这个项目的重要性
  • 你们想要招个怎样的人

总监面

  • 自我介绍
  • 项目介绍
  • 未来规划
  • 前端的了解

HR

  • 薪资、福利
  • 加班时间及相关制度
